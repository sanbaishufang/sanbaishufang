如何阅读一本书

发表于 2023-04-23 更新于 2023-05-07 分类于 读书笔记 阅读次数：
本文字数： 2.2k 阅读时长 ≈ 2 分钟

读书的意义

读书足以怡情，足以傅彩，足以长才。其怡情也，最见于独处幽居之时；其傅彩也，最见于高谈阔论之中；其长才也，最见于处世判事之际。培根在《谈读书》中如是说道。读书对我们是一种润物细无声的提升，它能够丰富我们的体验，帮助我们理解这个世界。

当我们想成功时，我们也许会读各种快速提升技能相关的书，这类书籍大多是成功学类的畅销书，从而忽略了像文学、哲学之类的非功利性的读书。如果我们读书都是抱着功利性的目的话，那我们接受的就只是一种成功主义的价值观。某一天我们从成功的山峰跌落时，是否还有勇气去面对人生的谷底呢？因此，我们也需要非功利性的阅读。每阅读一本书，都会不断地丰富我们人生的经验，知道我们为什么活在这个世界，知道任何伟大的人一生都不是平平顺顺的，他们也要遇到挫折，在阅读这些作品后，当我们身陷囹圄时，不偏执，不固陋，勇敢地与苦难和解。

应该读什么书

“世上无人人必读的书，只有在某时某地，某种环境，和生命中的某个时期必读的书。我认为读书和婚姻一样，是命运注定的或阴阳注定的。”（林语堂《生活的艺术》）譬如《诗经》这么经典，也不是人人都需要读，更何况很多人可能读不懂，每个人在生命某个时期都有必读的书，这些书也不尽相同，就像《大学》没有帮助王阳明找到格物致知的答案，《道德经》却帮曾国藩走出人生低谷。

“书是读不尽的，就读尽也是无用，许多书没有一读的价值。你多读一本没有价值的书，便丧失可读一本有价值的书的时间和精力，所以你须慎加选择。”（朱光潜《谈读书》）我们听过庄子的“吾生也有涯，而知也无涯”，知道知识是无穷无尽的，即使我们只读万分之一的书，这辈子也读不完，“以有涯随无涯，殆已”。哪些该读的书，在很多地方都有推荐书单，但哪些不该读的书却鲜有提及。在我看来，有两类的书应少读甚至不读，其一，讲述个人成功的鸡汤类畅销书，这类书缺乏充分的事实依据，通常很难复制，作者的观点大多局限于自身，不推荐阅读；其二，就某本书的主旨，你的认知水平已经高于作者，从这类书中已经理解不到任何事情，不推荐阅读。

读书的方法

王国维在《人间词话》中提到过人生的三大境界，读书的层次也类似于此，《如何阅读一本书》中艾德勒介绍阅读有四大层次。

第一层次的阅读：基础阅读，也可以说是初级阅读、基本阅读或者初步阅读。能识字即达到了基础阅读的层次。

第二层次的阅读：检视阅读，在较短的时间内，有系统的略读或粗读，了解书中讲了一件什么事，或者解决了一个什么样的问题。首先，通过看书名、序，在脑海中将这本书归类为某个特定的类型，在那个类型中，已经包含了哪些书。其次，通过研究目录页，对这本书的基本架构做概括性的理解。接着，如果有索引，则快速评估一下这本书涵盖了哪些议题的范围—这关系到作者的意图和态度。如果那是本包着书衣的新书，不妨读一下出版者的介绍，其中可能包含着书中的主旨。最后，通过上面的这些方法，可以足够判断是想要更仔细地读这本书，还是根本不想读下去了。如果还不够，则需要把书打开来，东翻翻西翻翻，念个一两段．有时候连续读几页，但不要太多，再去判断。

第三层次的阅读：分析阅读，全盘的阅读、完整的阅读，或者说是优质的阅读。在无限的时间里，最好也是最完整的阅读。

分析阅读的第一个阶段，找出一本书在谈些什么，有四个规则。第一个规则是对书籍进行分类，你一定要知道自己在读的是哪一类书，而且要越早知道越好。最好早在你开始阅读之前就先知道。比如，你需要知道它是虚构类的小说还是说明性的论述。第二个规则是使用一个单一的句子，或最多几句话（一小段文字）来叙述整本书的内容。首先，一位作者，特别是好的作者，会经常想要帮助你整理出他书中的重点。其次，要小心书的重点摘要可能不只有一个，通常没有绝对又唯一的说明。第三个规则可以说成是将书中重要篇章列举出来，说明它们如何按照顺序组成一个整体的架构。第四个规则是发现作者的意图，找出作者要问的问题。一本书的作者在开始写作时，都是有一个问题或一连串的问题，而这本书的内容就是一个答案，或许多答案。

分析阅读的第二个阶段，找出一本书到底在说什么，即诠释一本书的内容。第一个规则，抓住书中重要的单字，搞清楚作者是如何使用这个单字的。通过理解这个单字，与作者达成共识。第二个规则，从最重要的句子中抓出作者的重要主旨。第三个规则，找出作者的论述，重新架构这些论述的前因后果，以明白作者的主张。第四个规则，确定作者已经解决了哪些问题，还有哪些是未解决的。在未解决的问题中，确定哪些是作者认为自己无法解决的问题。

分析阅读的第三个阶段，像沟通知识一般地评论一本书，在读者理解作者所表达的含义后，读者应该能够与作者“辩论”。首先，先完整地了解一本书，不要急着开始批评。其次，不要争强好辩或盲目反对。最后，要为自己不同的意见找到理论基础，这样这个议题才不只是被说出来，而且会解释清楚。只有这样，才有希望解决这个问题。

第四层次的阅读：主题阅读。在做主题阅读时，会读很多书，并借助所阅读的书籍，架出一个中立的主题。针对这个主题，将不同作者的意见陈列在旁边，讨论之间的差异。

除了书籍外，阅读在今天已经有了更多的形式，比如短视频。从本质上来讲，阅读就是传播与接收的互动。无论哪种阅读形式，都要记住，读书不是为了炫耀，读书是为了培养你的智慧，在即将面临的人生苦难面前，给你带来更多的选项。
